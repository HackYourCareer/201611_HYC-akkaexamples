## Usage

### Prerequisites

- Download and install [Scala](http://www.scala-lang.org/download/)
- Download and install [SBT](http://www.scala-sbt.org/download.html)

### Run examples

- Clone repository
- Compile repository using

```
sbt compile
```

- Start examples:
```
sbt run
```

and select which example will be started.
